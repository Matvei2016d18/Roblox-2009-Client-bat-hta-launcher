# Roblox 2009 Launcher (BAT + HTA)

A simple launcher for classic ROBLOX 2009 places.

## 🚀 How to use

1. Download all files into one folder.
2. Run `1 start MAP.bat` OR open `RobloxSite.hta`.
3. If you use `.bat` – enter the map name.  
   If you use `.hta` – click the `Play This` button next to the map.
4. When **Roblox Studio** opens, paste the command from your clipboard (Ctrl + V) into the **Command** field and press Enter.
5. Your character will appear — you can play now! 🎮

## 🗺️ Where to get maps
- [TEMPORTAL](https://temportal.neocities.org/) – 2006–2008 maps.

## 📌 Note
- The command is already copied to your clipboard when you run the `.bat` file or click `Play This` in the `.hta` interface.
- This project was made to preserve the old ROBLOX vibe — no monetization, just classic fun.

## 🛠️ Files
- `1 start MAP.bat` – command line launch
- `RobloxSite.hta` – GUI launcher
- `maps to play/` – folder for `.rbxl` maps
